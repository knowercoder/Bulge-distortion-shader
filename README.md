# Bulge-distortion-shader
Bulge image distortion shader for threejs

Live demo: https://knowercoder.github.io/Bulge-distortion-shader/

![Bulgeimagedistortionshader](https://github.com/user-attachments/assets/94085cf9-87cf-4d71-a909-af56e18bc1a5)

in the script.js file:
- change the image at line 40
- change the distortion scale and radius at lines 46, 47
- The image is being rendered on a plane mesh. you can change the image size, scale and rotation at lines 92, 93, 94
